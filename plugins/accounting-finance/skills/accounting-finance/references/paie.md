## Du brut au net

| Étape | Calcul | Résultat |
|---|---|---|
| 1. **Salaire brut** | Base + HS + primes + avantages nature | Brut |
| 2. **Cotisations salariales** | Vieillesse + AGIRC-ARRCO + CEG + CET + prévoyance + mutuelle | Total cot. salariales |
| 3. **CSG/CRDS** | 98,25 % brut x (6,80 % + 2,40 % + 0,50 %) | Total CSG + CRDS |
| 4. **Net imposable** | Brut - cot. salariales déduc. - CSG déduc. + CSG non déduc. + CRDS + mutuelle patronale | Net imposable |
| 5. **Net avant PAS** | Brut - total cot. salariales - CSG/CRDS - avantages nature | Net à payer avant PAS |
| 6. **PAS** | Net imposable x taux PAS | Montant PAS |
| 7. **Net à payer** | Net avant PAS - PAS + remboursements frais | Net versé |

## Réductions et exonérations

### Réduction générale (ex-Fillon) — Art. L. 241-13 CSS

| Paramètre | Valeur |
|---|---|
| **Bénéficiaires** | Employeurs secteur privé, tous salariés |
| **Condition** | Rémunération <= 1,6 SMIC annuel brut |
| **Cotisations** | SS, chômage, AGIRC-ARRCO |
| **Coeff. T max** | 0,3194 (>= 50 sal.) ou 0,3234 (< 50 sal.) |
| **Formule C** | C = (T / 0,6) x [(1,6 x SMIC annuel / Rém. annuelle) - 1] |
