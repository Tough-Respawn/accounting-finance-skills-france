# Finance d'entreprise — Analyse financière et évaluation

> L'analyse financière n'est pas réglementée en tant que telle mais s'appuie sur des méthodologies établies (OEC, Banque de France, AMF). Les sociétés commerciales sont tenues d'établir un rapport de gestion (C. com. art. L. 232-1) incluant une analyse objective de l'évolution des affaires, des résultats et de la situation financière.

## Bilan fonctionnel

### Structure

| Cycle | Emplois (Actif) | Ressources (Passif) |
|-------|-----------------|---------------------|
| **Stable** | Emplois stables bruts : immob. incorporelles, corporelles, financières (valeurs brutes) | Ressources stables : CP + amort. et dépr. + provisions + dettes financières > 1 an |
| **Exploitation** | ACE : stocks, créances clients, avances versées, CCA exploitation | PCE : dettes fournisseurs, avances reçues, dettes fiscales/sociales exploitation, PCA exploitation |
| **Hors exploitation** | ACHE : créances diverses, CCA hors exploitation | PCHE : dettes sur immob., IS, autres dettes HE, PCA HE |
| **Trésorerie** | TA : disponibilités, VMP | TP : CBC, soldes créditeurs de banque |

### Grands équilibres

| Indicateur | Formule | Interprétation | Norme |
|-----------|---------|---------------|-------|
| **FRNG** | Ressources stables - Emplois stables bruts | Excédent de financement long terme | > 0 |
| **BFRE** | ACE - PCE | Besoin lié au cycle d'exploitation | Selon secteur |
| **BFRHE** | ACHE - PCHE | Besoin hors cycle courant | Faible |
| **BFR** | BFRE + BFRHE | Besoin global de financement | Variable |
| **Trésorerie nette** | FRNG - BFR = TA - TP | Résultante | > 0 souhaitable |

### Retraitements du bilan fonctionnel

| Poste | Retraitement | Justification |
|-------|-------------|---------------|
| Amortissements/dépréciations | Ajoutés aux ressources stables | Source de financement interne |
| CBC | Trésorerie passive (hors dettes fin.) | Financement très court terme |
| EENE | ACE + trésorerie passive | Créance mobilisée = financement CT |
| Crédit-bail | Emplois stables + ressources stables | Location-financement assimilable à investissement |
| CCA/PCA | Ventilés exploitation/HE | Selon la nature |
| Écarts de conversion actif | Réintégrés dans les postes | Neutralisation du change |
| Primes de remboursement | Éliminées actif et dettes | Charge financière à étaler |
| Intérêts courus | PCHE | Charge CT |
| Capital non appelé | Éliminé actif et CP | Non mobilisable |

**Keywords :** bilan fonctionnel, FRNG, BFR, BFRE, trésorerie nette, fonds de roulement, emplois stables, ressources stables

## Effet de levier financier

### Formule

Re = Réco + (Réco - i) x D/CP

| Variable | Signification |
|----------|--------------|
| Re | Rentabilité financière (ROE) |
| Réco | Rentabilité économique (EBIT net d'IS / actif économique) |
| i | Coût moyen de la dette net d'IS |
| D | Dettes financières |
| CP | Capitaux propres |
| D/CP | Bras de levier |
| (Réco - i) | Différentiel de levier |

Formule après impôt : Re = [Réco + (Réco - i) x D/CP] x (1 - t), avec t = 25 %

### Interprétation

| Situation | Condition | Effet sur ROE |
|-----------|-----------|--------------|
| Levier positif | Réco > i | L'endettement augmente le ROE |
| Levier nul | Réco = i | Pas d'impact |
| Levier négatif (effet de massue) | Réco < i | L'endettement diminue le ROE |

### Décomposition DuPont

ROE = (Résultat net / CA) x (CA / Total actif) x (Total actif / CP)
    = Marge nette x Rotation des actifs x Levier financier

| Composante | Levier d'amélioration |
|-----------|----------------------|
| Marge nette | Maîtrise des coûts, pouvoir de prix |
| Rotation des actifs | Gestion stocks, créances |
| Levier financier | Structure de financement |

**Keywords :** effet de levier, ROE, rentabilité financière, rentabilité économique, endettement, DuPont

## Évaluation d'entreprise

### DCF (Discounted Cash Flows)

#### Free Cash Flows

| Élément | Formule |
|---------|---------|
| EBIT (résultat d'exploitation) | Produits - Charges d'exploitation |
| - Impôt normatif | EBIT x taux IS (25 %) |
| = NOPAT | EBIT x (1 - t) |
| + Amortissements | Charges non décaissables |
| - Variation du BFR | Augmentation = emploi |
| - CAPEX | Acquisitions d'immobilisations |
| **= FCF** | Flux disponible actionnaires + créanciers |

#### CMPC / WACC

| Composante | Formule |
|-----------|---------|
| CMPC | Ke x CP/(CP+D) + Kd x (1-t) x D/(CP+D) |
| Ke (MEDAF) | Rf + beta x (Rm - Rf) |
| Beta désendetté | beta_e / [1 + (1-t) x D/CP] (Hamada) |
| Beta réendetté | beta_u x [1 + (1-t) x D/CP] |

Ke : coût des CP ; Kd : coût de la dette avant IS ; Rf : taux sans risque (OAT 10 ans) ; Rm-Rf : prime de risque marché (4-7 % France)

#### Valeur terminale

| Méthode | Formule |
|---------|---------|
| Gordon-Shapiro | VT = FCF(n+1) / (CMPC - g), g = 1,5-2,5 % |
| Multiple de sortie | VT = EBITDA(n) x multiple sectoriel |

#### Valeur d'entreprise

| Étape | Formule |
|-------|---------|
| VE | Somme FCF(t)/(1+CMPC)^t + VT/(1+CMPC)^n |
| Valeur des CP | VE - DFN + actifs hors exploitation |
| Valeur par action | Valeur CP / nombre d'actions |

### Multiples

| Multiple | Formule | Utilisation | Fourchette type |
|----------|---------|-------------|----------------|
| EV/EBITDA | VE / EBITDA | Le plus courant ; neutralise amort. et structure | 5-12x |
| EV/EBIT | VE / EBIT | Plus conservateur | 8-15x |
| EV/CA | VE / CA | Sociétés en perte ou forte croissance | Variable |
| PER (P/E) | Cours / BPA | Fondé sur le résultat net | 10-25x |
| P/B (Price/Book) | Capitalisation / CP comptables | Banques, assurances, immobilier | 0,5-2x |
| EV/FCF | VE / FCF | Flux de trésorerie | Variable |
| PEG | PER / taux croissance BPA | Ajusté de la croissance | PEG < 1 = sous-évalué |
| EV/EBITDAR | VE / (EBITDA + loyers) | Distribution, hôtellerie | Variable |

### Méthode patrimoniale

| Concept | Formule |
|---------|---------|
| ANC | Total actif - Total dettes - Actifs fictifs |
| ANCC | ANC + Plus-values latentes - Moins-values +/- ID |
| ANR | ANCC ajusté de la fiscalité latente |
| Goodwill (patrimonial) | Valeur de rendement - ANCC |
| Rente de goodwill | (BN récurrent - Rémunération normale ANCC) actualisée |
| Valeur de rendement | BN récurrent / taux de capitalisation |

**Keywords :** évaluation, DCF, CMPC, WACC, MEDAF, CAPM, bêta, valeur terminale, multiples, EV/EBITDA, PER, ANR, goodwill, FCF

## Choix d'investissement

| Critère | Formule | Règle de décision | Limites |
|---------|---------|-------------------|---------|
| **VAN** | -I0 + Somme CFt/(1+k)^t | VAN > 0 = rentable | Sensible au taux |
| **TRI** | Taux k tel que VAN = 0 | TRI > coût du capital | TRI multiples possibles |
| **Délai de récupération** | Nb périodes pour CF cumulés >= I0 | Plus court = moins risqué | Ignore flux après |
| **Délai actualisé** | Nb périodes pour CF actualisés >= I0 | Plus court = meilleur | Ignore flux après |
| **IP** | Somme CFt/(1+k)^t / I0 | IP > 1 = rentable | Peut contredire VAN |

### Flux d'un projet

| Élément | Calcul | Période |
|---------|--------|---------|
| I0 | Prix + frais + augmentation BFR initial | Année 0 |
| CFt | EBE projet x (1-t) + Amort. x t | Années 1 à n |
| Variation BFR | Augmentation = emploi | Chaque année |
| Valeur résiduelle | Prix cession net IS + récupération BFR | Année n |

### Taux d'actualisation

| Situation | Taux |
|-----------|------|
| Risque identique | CMPC |
| Risque différent | CMPC ajusté (bêta sectoriel) |
| 100 % CP | Ke |

**Keywords :** VAN, TRI, délai de récupération, indice de profitabilité, investissement, CMPC, CAPEX

## Politique de financement

### Sources

| Source | Type | Coût | Avantages | Inconvénients |
|--------|------|------|-----------|--------------|
| Autofinancement | Interne | Ke (opportunité) | Pas de dilution | Limité |
| Augmentation de capital | CP | Ke | Renforce fonds propres | Dilution |

### Théories de la structure du capital

| Théorie | Auteurs | Proposition |
|---------|---------|------------|
| Modigliani-Miller (sans impôt) | MM 1958 | Valeur indépendante de la structure |

| Trade-off | Kraus & Litzenberger 1973 | Ratio optimal = équilibre éco. fiscale vs coûts faillite |
| Pecking order | Myers & Majluf 1984 | Autofinancement > dette > CP |
| Signaling | Ross 1977 | Endettement = signal de confiance |
| Agency | Jensen & Meckling 1976 | Dette discipline les dirigeants |
